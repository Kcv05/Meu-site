Projeto 2: GameQuiz - Jogo de Perguntas

Desenvolvi um jogo de perguntas e respostas com interface moderna e animações suaves. O projeto inclui sistema de pontuação, diferentes categorias de perguntas e design responsivo para mobile. Foi um desafio interessante implementar a lógica do jogo e criar uma experiência de usuário envolvente.

Tecnologias: HTML5, CSS3, JavaScript, LocalStorage

Projeto 3: WeatherApp - App do Clima

Criei um aplicativo de previsão do tempo com integração a API externa. O app mostra temperatura atual, previsão para os próximos dias e informações detalhadas como umidade e velocidade do vento. Implementei geolocalização para detectar automaticamente a cidade do usuário.

Tecnologias: HTML5, CSS3, JavaScript, API REST, Geolocalização

Projeto 4: TaskManager - Gerenciador de Tarefas

Desenvolvi um gerenciador de tarefas completo com funcionalidades de adicionar, editar, excluir e marcar tarefas como concluídas. O projeto inclui persistência de dados no navegador e interface intuitiva com drag and drop para reorganizar tarefas.

Tecnologias: HTML5, CSS3, JavaScript, LocalStorage, Drag & Drop API

Projeto 5: MusicPlayer - Player de Música

Criei um player de música com design moderno e funcionalidades avançadas como controle de volume, playlist personalizada e visualização de letras. O projeto demonstra habilidades em manipulação de áudio e criação de interfaces interativas.

Tecnologias: HTML5, CSS3, JavaScript, Web Audio API, Canvas
